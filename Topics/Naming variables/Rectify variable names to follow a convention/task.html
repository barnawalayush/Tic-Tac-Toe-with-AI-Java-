Rectify variable names to follow a convention

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Given a string containing a list of variable names, rectify them according to Java variable naming conventions. The input string includes variable names separated by a comma. Each variable name is made using alphabets, digits and underscores. If a variable doesn't follow the convention (starts with a letter, doesn't start with a digit, uses only alphanumeric characters, and underscores), adjust it. If a variable starts with a digit, prepend 'var'. If a variable includes other than alphanumeric characters or underscores, replace them with an underscore. Output the rectified string, preserving the original order and the commas.
Sample Input:
123alpha, beta, gamma@123, $$$

Sample Output:
var123alpha, beta, gamma_123, ___


Sample Input:
_alpha, 123beta, gamma$123, ___

Sample Output:
_alpha, var123beta, gamma_123, ___



Memory limit: 256 MB
Time limit: 5 seconds
